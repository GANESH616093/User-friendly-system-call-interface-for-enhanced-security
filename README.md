# User-friendly-system-call-interface-for-enhanced-security
web based secure system call simulation
This project simulates a secure system call interface using HTML, CSS, and JavaScript.

## Features
- User authentication
- Policy-based access control
- System call simulation
- Logging and reporting

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open index.html in browser
2. Login and test system calls
